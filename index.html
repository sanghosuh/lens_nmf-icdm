Background

Topic Modeling

Two approaches


1. Probabilistic (e.g. LDA)

2. Matrix factorization (e.g. NMF)

Nonnegative Matrix Factorization (NMF)


  • Nonnegative means all the elements in the matrix has to be nonnegative

  • In other words, anything that is negative is round up to 0
  • It improves(?) interpretability (i.e. eliminates the need to interpret negative values)
  • Compared to probabilistic model, the results are NOT random
Motivation
  • We want to discover topics, which summarize a large volume of contents(corpus)
  • Current topic modeling algorithms provide global topics
  • They do NOT provide meaningful information as they are general information we already know

Can we create an algorithm that provides local, focused topics of interest for more informative topic discovery?

How can we find local topics?

Line of Reasoning


  • NMF produces W, H where we use columns of W as topics of keywords

Line of Reasoning


Residual matrix, R, represent unexplained topics

Line of Reasoning


  • A single iteration of NMF does NOT provide quality local topics
  • Let's use ensemble model based on gradient boosting framework to iteratively boost up the values of local topics so that they become major topics

Proposed solution

Intuition


Intuition


  • Residual matrix after performing NMF can be regarded as under-represented topics
  • Thus, if we utilize residual matrix well, we can discover local topics
  • Construct an ensemble of NMF models based on a gradient boosting framework
